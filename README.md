# CanineAI - Advanced AI Dog Monitoring Research Project

CanineAI is a cutting-edge research project dedicated to improving the health and well-being of domestic dogs through state-of-the-art Artificial Intelligence. Our platform provides real-time monitoring and advanced diagnostics to empower dog owners with data-driven insights.

## 🚀 Key Research Pillars

Our research is divided into three core components, each led by dedicated team members:

1.  **Breed Detection**: Utilizing advanced deep learning models to accurately identify dog breeds from multi-modal data.
2.  **Growth Monitoring**: A comprehensive system to track physical development metrics against breed-specific benchmarks.
3.  **Skin Disease Detection & Treatment**: An AI-driven diagnostic tool to identify skin conditions and provide immediate care recommendations.

## 🛠️ Technology Stack

- **Frontend**: React.js, Tailwind CSS, Framer Motion (for animations)
- **Icons**: Lucide React
- **Email Service**: EmailJS
- **Backend**: Node.js (Express)
- **Animations**: Framer Motion

## 📦 Getting Started

### Prerequisites
- Node.js (v16+)
- npm

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Kavinilavan1221/RP-Website.git
    cd RP-Website
    ```

2.  **Setup Client**:
    ```bash
    cd client
    npm install
    npm run dev
    ```

3.  **Setup Server**:
    ```bash
    cd ../server
    npm install
    node index.js
    ```
